# Security Transparency Report Template

Reporting period: YYYY-QN

## 1. Executive Summary

- Security posture changes this period
- Major risks accepted, mitigated, or retired

## 2. Audit & Review Status

- External audits initiated/completed
- Findings by severity and remediation status
- Re-audit trigger events and outcomes

## 3. Vulnerability Handling

- Reports received / validated / fixed
- Median response and remediation times
- Coordinated disclosures published

## 4. Release Integrity

- Releases shipped
- Signed tags/checksums/attestations status
- SBOM publication status

## 5. Protocol & Compatibility Changes

- Protocol changes and compatibility impact
- Migration notes issued

## 6. Privacy & Metadata Posture

- Changes to metadata exposure assumptions
- Traffic-analysis mitigation updates

## 7. Assurance Scorecard Delta

- Updated scores and rationale versus prior period

## 8. Next-Period Commitments

- Planned hardening work
- Audit/re-audit milestones
